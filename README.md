# chatbot
🤖 Groq AI Chatbot
A simple AI Chatbot Web Application built using Flask and Groq API (LLaMA model).
Users can send messages through a web interface and receive AI-generated responses instantly.

---

## 🚀 Features
- 💬 Real-time chatbot interaction
- ⚡ Fast AI responses using Groq API
- 🌐 Simple and clean web interface
- 🔒 Secure API key using .env file
- 🧠 Handles dynamic user queries

---

## 🛠 Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- API: Groq AI (LLaMA 3.1 model)
- Library: Requests
